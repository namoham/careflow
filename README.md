# CareFlow

**Supported Living Management Application**

A Flutter-based mobile application for managing supported living agencies and group homes.

---

## Overview

CareFlow streamlines daily operations in supported living environments through:
- Staff scheduling and shift management
- Client tracking and care planning
- Incident reporting with severity classification

---

## Features

| Module | Functionality |
|--------|---------------|
| Dashboard | Real-time statistics and shift overview |
| Clients | Client directory with room assignments |
| Staff | Staff management with role-based access |
| Shifts | Schedule management and status tracking |
| Incidents | Incident logging with severity levels |

---

## Technical Stack

- **Framework:** Flutter
- **Language:** Dart
- **Design:** Material Design 3
